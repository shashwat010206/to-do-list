How to Use

  1. Enter a task title and optional description.
  2. Select a priority, category, and due date.
  3. Click the "Add Task" button to add it to the list.
  4. Use the buttons in each task to complete, edit, or delete.
  5. Use the search box and filters to quickly find tasks.
